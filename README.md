# CodeAlpha Banking System

## Description
This is a simple Banking System developed in C++. It allows users to deposit money, withdraw money, and check their account balance.

## Features
- Deposit Money
- Withdraw Money
- Check Account Balance
- Menu-driven Program
- Object-Oriented Programming (Classes)

## Technologies Used
- C++
- Visual Studio Code

## How to Run
1. Compile the program using a C++ compiler.
2. Run the executable.
3. Choose the required banking operation.
